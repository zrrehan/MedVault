# MediVault 💊  
**Medicine Delivery System**

---

## Project Overview

MediVault is a full-stack medicine delivery system designed to simplify how users order medicines online and receive them at their doorstep. The platform connects customers, sellers (pharmacies), and admins in a streamlined workflow where ordering, payment, and delivery management are handled efficiently.

Unlike a traditional e-commerce platform, MediVault focuses on delivery flow — ensuring medicines move smoothly from seller listings to admin-controlled delivery completion.

## Features

### 👤 Customer
- Register and login securely  
- Browse available medicines  
- View detailed medicine information  
- Add medicines to cart  
- Place orders with integrated online payment  
- Track order status (Pending → Delivered)  
- Manage personal profile  

---

### 🏪 Seller (Pharmacy)
- Register and login as a seller  
- Add new medicines  
- Update or delete existing medicines  
- Manage available stock  
- Focus only on listing products (no delivery control)  

---

### 🛠️ Admin
- Full control over the system  
- Manage all users (customers & sellers)  
- View and monitor all orders  
- Control delivery flow by updating order status  
  - Pending  
  - Delivered  
- Oversee platform activity and ensure smooth operations  

---

## Key Highlights

- 💳 Integrated payment system for secure transactions  
- 🚚 Delivery lifecycle fully controlled by admin  
- 🔒 Role-based authentication and authorization  
- 📦 Clean separation of responsibilities:
  - Seller → Product management  
  - Admin → Delivery management  
- ⚡ Fast and responsive user experience  

---

## Note

This system is designed for handling **OTC (over-the-counter) medicines only**.
